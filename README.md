# Learnquest

An interactive mobile-friendly ed-tech educational game built to make learning fun and engaging.

## 🚀 Live Demo
Play the game here: [v0-learning-quest.vercel.app](https://vercel.app)

## 🎮 Game Features
* **Adventure Map**: Embark on educational quests across different stages.
* **Counting Castle**: Master mathematics and counting skills.
* **Word Woods**: Enhance vocabulary and reading levels.
* **Color Canyon & Globe Grotto**: Explore interactive art and geography challenges.
